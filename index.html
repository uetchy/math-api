Math API

Place LaTeX Math equation on anywhere as <img> tag.

Usage

https://math.now.sh?from=[LaTeX Math Equation]

Block notation

https://math.now.sh?from=\sum^{N}_{i}x_i

Inline notation

https://math.now.sh?inline=\sum^{N}_{i}x_i

PNG support

https://math.now.sh?from=\sum^{N}_{i}x_i.png

HTML Example

<img src="https://math.now.sh?from=\log\prod^N_{i}x_{i}=\sum^N_i\log{x_i}" />

<img src="https://math.now.sh?inline=\log\prod^N_{i}x_{i}=\sum^N_i\log{x_i}" />

![Equation 1](https://math.now.sh?from=\LaTeX)
